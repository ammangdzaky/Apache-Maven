PENGENALAN BUILD AUTOMATION
    Build automation adalah proses meng-otomatisasi tahapan pembuatan software dan hal-dal yang
    berhubungan dengannya, seperti: kompilasi source kode menjadi binary code, mem-package
    binary code menjadi distribution file, membuat dokumentasi, menjalankan automated test sampai
    manajement depedency


CONTOH BUILD AUTOMATION
    - Apache Maven
    - Apache Ivy
    - Gradle


PENGENALAN APACHE MAVEN
    - Apache Maven adalah salah satu build automation yang free dan open source
    - Apache menggunakan XML untuk mendefenisikan build scriptnya
    - Apache Maven saat ini sangat populer di kalangan Developer Java
    - Apache Maven menggunakan JVM sebagai pondasi dasar


TEKNOLOGI YANG DIDUKUNG APACHE MAVEN
    - Java
    - Kotlin
    - Groovy
    - Scala
